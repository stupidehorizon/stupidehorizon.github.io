一分钟快速起一个MongoDB数据库

2017-03-17 • 标签: MongoDB

###一分钟快速起一个MongoDB数据库

1、数据库安装

此处不多说直接百度官网找到对应自己电脑的版本下载傻瓜式安装。

2、在系统 高级设置 环境变量path中添加已安装MongoDB的bin文件路径

比如我添加的是 D:\MongoDB\Server\3.4\bin

3、在任意盘下新建数据库存放文件夹

比如我在D盘下创建了MongoDB-data文件，并在里面创建了data文件，然后再在data里面创建了db文件

4、现在正式开始启动数据库

首先明确MongoDB bin 文件中三个可执行文件的区别

bin/mongod - The database process.
bin/mongos - Sharding controller.
bin/mongo - The database shell (uses interactive javascript).
不清楚这三者的区别是刚开始不知道怎么起数据库的主要原因

简而言之 通过mongod建立一个数据库并指定一个端口等待连接。
通过 mongo(MoNgoDB shell)来连接并且操作数据库。

  • 打卡命令行输入 mongod -port 28008 -dbpath D:\MongoDB-data\data\db
    *//第二种方法起一个数据库 mongod –config D:\MongoDB\Server\3.4\bin\mongod_config.txt
  • 再打开一个命令行输入 mongo localhost:28008

不出意外数据库启动并连接成功！

#####5、 关闭数据库
在使用 mongo 的命令行中输入

use admin
db.shutdownServer()

####6、MongoDB shell脚本编程

  • 在命令行中使用 –eval 执行js字符串或文件。
    > mongo local --eval "printjson(db.getCollectionNames())"
  • 在MongoDB shell中使用 load()加载js文件。
    >load("name.js")

#####7、管理用户账户
MongoDB能正常运行后，首先应该想到的事是添加用户，接下来我们讨论如何使用MongoDB shell来管理用户账户。

要创建用户账户，首先切换到目标数据库，再使用addUser()创建用户账户。
use local
db.adduser({
user:”testUser”,
pwd:”test”,
roles:[“readWrite”,”dbAdmin”]
})
