ES6入门之let,const

2017-08-13 • 标签: javascript

看完阮一峰的ES6标准入门也有一个月了，因为我看书的时候习惯用笔做笔记，所以也有记了很多笔记。加之最近在公司实习基本上也没有用到ES6，所以现在想把笔记整理出来，学过没有用的东西可能很快就会忘记了所以算是一次复习吧。

let

  1. let申明的变量只在let命令所在的代码块内有效；

  2. 不存在变量提升；

  3. 暂时性死区

    只要块级作用域内存在let命令，它所声明的变量就绑定在这个区域，不受外界影响；

  4. 不允许重复声明

    const

  5. const用来声明常量，一旦申明，其值就不能改变；
  6. 一旦申明就必须立即初始化（赋值）；
  7. 和let一样只在所声明的代码块内有效；
  8. 和let一样不可重复声明；
  9. 对于复合类型的变量，const命令只保证变量指向的地址不变，并不保证该地址的数据不变。
