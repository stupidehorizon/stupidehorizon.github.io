字符和数子类型的快速转换

2017-08-13 • 标签: javascript

在实际工作中经常会遇到类型转换，因为javascript隐式的类型转化。所以很多时候我们不注意可能就会出现一些问题。比如，我以为Boolean(“0”) ==false。实际上：

Boolean(“0”) //true
Boolean(+”0”) //false
所以为了不导致错误，我们最好手动的进行类型转换，比如在字符串前面加“+”号可将字符串转化为数字！

+”aaaa” //NaN
+1 //1
+”1a” //NaN
+”1” //1
+true //1
+false //0
同理其他类型转化为字符串可以在前面加引号：

+”1” //1
‘’+1 //“1”
‘1’+1 //“11”
‘’+0 //“0”
‘’+true //“true”
‘’+false //“false”
