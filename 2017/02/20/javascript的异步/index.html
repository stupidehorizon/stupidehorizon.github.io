javascript的异步

今天在网上看见一个题
求
for (var i = 0; i < 10; ++i) {
setTimeout(function () {console.log(i)}, 0);
}
得打印结果。答案是10，10，10，…,10。10个10。然后我百度了关于javascript的异步的知识看了阮一峰的博客Javascript异步编程的4种方法 发现这不是其中任何一种异步编程方法啊 最后又看了深入理解JavaScript编程中的同步与异步机制。才发现setInterval,setTimeout,里面的函数本身
就是回调函数，当异步函数执行时，回调函数会被压入一个队列。JavaScript引擎直到异步函数执行完成后，才会开始处理事件循环。所以说上面的函数每循环一次都是往队列里面压入一个待执行函数，而循环执行完后才会，，执行压入的10个函数，而此时i已经等于10。
