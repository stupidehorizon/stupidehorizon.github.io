关于js == 操作符的隐式类型转换

2017-09-28 • 标签: javascript

众所周知当在js中使用 == 操作符来比较两个不同类型的数据时会进行隐式的类型转换，这个在给我们带来方便的同时也带来了很大的隐患。所以理解js的隐式类型转换十分有必要。

  1. == 两边皆为字符串

    从左到右依次比较每一个字符的ASCII码，若全部一样，则返回true，否则返回false。

  2. == 两边为对象

    只有两个对象指向的是同一个对象时为true;
