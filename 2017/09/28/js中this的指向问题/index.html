js中this的指向

2017-09-28 • 标签: javascript

该开始学js的时候可能总是会被js中this的指向搞晕；其实我们只要记住js中的this指向完全取决于函数的调用方式，this 绑定的对象即函数执行的上下文环境。就很好理解了。

  • 情况1：如果一个函数中有this，但是它没有被上一级的对象所调用，那么this指向的就是window(非严格模式，在严格版中的默认的this不再是window，而是undefined)
  • 情况2：如果一个函数中有this，这个函数有被上一级的对象所调用，那么this指向的就是上一级的对象。
  • 情况3：如果一个函数中有this，这个函数中包含多个对象，尽管这个函数是被最外层的对象所调用，this指向的也只是它上一级的对象
