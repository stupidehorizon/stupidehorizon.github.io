实用的排序算法——归并排序

2017-05-01 • 标签: javascript

  • 归并排序

    javascript的Array类中定义了一个sort函数用来排序数组。Mozilla Firefox使用就使用了归并排序来实现这个方法。Chrome使用的是快速排序。
    归并排序是一种分治算法，将原始数组切分为较小的数组，直到每个数组只有一个元素，接着将小数组归并为较大的数组。

结果为[1, 2, 3, 4, 4, 5, 6]

现在我们用面向对象的方式来改写这个排序方法

得到结果[1, 2, 2, 4, 5, 6]
