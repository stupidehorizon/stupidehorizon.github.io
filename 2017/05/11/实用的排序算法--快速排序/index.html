实用的排序算法——快速排序

2017-05-11 • 标签: javascript

  • 快速排序

    javascript的Array类中定义了一个sort函数用来排序数组。Mozilla Firefox使用就使用了归并排序来实现这个方法。Chrome使用的是快速排序。
    快速排序也是一种分治算法，将原始数组切分为较小的数组，但是它没有像归并那样把他们分隔开。

总结

快速排序也是一种分治算法，不同于归并排序的是它失踪在操作原数组。一次排序后确保index前的值比index后的值小。然后再分别排序前面和后面，直到left=index-1;index=right，此次排序完毕。
